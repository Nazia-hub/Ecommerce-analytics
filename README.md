# 🛒 E-Commerce Data Analysis Project

## 📌 Overview
This project focuses on analyzing an e-commerce dataset using **PostgreSQL** and **Power BI** to extract meaningful business insights related to sales, customers, products, delivery, and reviews.

---

## 🎯 Objectives
- Analyze customer purchasing behavior
- Identify top-performing products
- Evaluate delivery efficiency
- Understand customer satisfaction using reviews
- Track revenue trends

---

## 🛠️ Tools & Technologies
- PostgreSQL (Database & SQL)
- Power BI (Data Visualization)
- GitHub (Version Control)

---

## 📂 Dataset Description
The project includes the following datasets:

- Customers
- Orders
- Order Items
- Payments
- Products
- Reviews
- Shipments

---

## 🧩 Database Schema
Relationships:
- Customers → Orders
- Orders → Order Items
- Orders → Payments
- Orders → Shipments
- Products → Order Items
- Products → Reviews
- Customers → Reviews

---
